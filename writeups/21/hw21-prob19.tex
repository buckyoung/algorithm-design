\centerline{\textbf{Problem 19}}
\leavevmode
(6 points) In the disjoint paths problem the input is a directed graph G and pairs ($s_1$, $t_1$), ...,($s_k$, $t_k$) of vertices. The problem is to determine if there exist a collection of vertex disjoint paths between the pairs of vertices (from each $s_i$ to each $t_i$). Show that this problem is NP-hard by a reduction from the 3SAT problem.
\\ \\
\textbf{Input:} Directed graph G / Vertex pairs ($s_1$, $t_1$), ...,($s_k$, $t_k$)
\\ \\ \textbf{Output:} 1 if there is a collection of vertex disjoint paths between the inputted vertex pairs, 0 otherwise
\\ \\ \textbf{Theorem:} Problem Vertex Disjoint Paths (VDP) is NP-Hard
\\ \\ \centerline{\textbf{3SAT $\leq$ VDP}}
\\
\\Program 3SAT ( formal F ):
\begin{algorithmic}[1]
\State G = a graph constructed of each vertex pair and "gadgets" for each clause in F
\State Vertex Pairs ($s_1$, $t_1$), ...,($s_k$, $t_k$) = a pair of vertices for each variable in the formula F
\State \textbf{return} VDP( G, vertex pairs )
\end{algorithmic}
